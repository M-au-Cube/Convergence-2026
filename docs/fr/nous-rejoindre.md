# Nous rejoindre

Vous souhaitez en savoir plus sur Convergence ESG, devenir partenaire ou contribuer à nos projets ? Remplissez le formulaire ci-dessous.

<form class="contact-form" action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
  <label for="name">Nom / Name *</label>
  <input type="text" id="name" name="name" required placeholder="Votre nom">

  <label for="email">E-mail *</label>
  <input type="email" id="email" name="email" required placeholder="votre@email.com">

  <label for="status">Statut *</label>
  <select id="status" name="status" required>
    <option value="" disabled selected>Sélectionnez votre statut</option>
    <option value="entreprise">Entreprise</option>
    <option value="organisation-publique">Organisation publique</option>
    <option value="consommateur">Consommateur</option>
  </select>

  <label for="organization">Organisation (facultatif)</label>
  <input type="text" id="organization" name="organization" placeholder="Nom de votre entreprise ou organisation">

  <label for="message">Message *</label>
  <textarea id="message" name="message" rows="6" required placeholder="Décrivez votre intérêt pour Convergence ESG…"></textarea>

  <input type="hidden" name="_subject" value="[Convergence ESG] Nouveau contact">
  <input type="text" name="_gotcha" style="display:none" tabindex="-1" autocomplete="off">

  <button type="submit">Envoyer</button>
</form>

!!! warning "Configuration du formulaire"
    Ce formulaire utilise [Formspree](https://formspree.io/) pour l'envoi des messages sur un site statique. Remplacez `YOUR_FORM_ID` dans `docs/fr/nous-rejoindre.md` (et la version anglaise) par votre identifiant Formspree, ou modifiez l'attribut `action` du formulaire.
